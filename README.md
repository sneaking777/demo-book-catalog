# borodina-book-catalog

Тестовое задание для **АНО ДПО «НАМО им. Н.А. Бородина»** (Тюмень) — каталог книг на Yii2 + MySQL.

## Стек

- PHP 8+
- Yii2 (basic template)
- MySQL/MariaDB
- Composer

## Что должно работать

- Просмотр каталога книг и авторов — без авторизации.
- CRUD книг и авторов — для авторизованного пользователя.
- Подписка гостя на нового автора с уведомлением (бонус — SMS через `smspilot.ru`, ключ «эмулятор»).
- Отчёт «топ-10 авторов по числу книг за указанный год» — без авторизации.

Полный текст задания — `TASK.md`.

## Установка (будет дополнено после инициализации шаблона)

```bash
composer install
cp config/db.example.php config/db.php   # заполнить реквизиты MySQL
php yii migrate
php -S localhost:8080 -t web
```

Тестовые учётные данные авторизованного пользователя — будут добавлены вместе с миграцией-сидом.

## Лицензия

Тестовое задание, без коммерческого использования.
